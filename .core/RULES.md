# Brolier 360 Coding Rules

## Python Backend
- Strictly use type hints for all function arguments and return types.
- Follow PEP 8 guidelines.
- Handle database sessions safely, closing them or using dependency injection (Depends) properly.

## React Frontend
- Write all components in TypeScript (`.tsx`).
- Use Tailwind CSS v3 for all styling. Do not write custom CSS unless absolutely necessary.
- Use Shadcn UI for all standard components (buttons, inputs, dialogs).
- Functional components only with React Hooks.
